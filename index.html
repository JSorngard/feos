    Welcome to FeOs¶

    FeOs is a framework for thermodynamic equations of state and classical functional theory. It is written in Rust with an interface for Python.

    • You can use FeOs with implemented equations of state to compute

      • thermodynamic properties,

      • phase equilibria,

      • critical points,

      • dynamic properties via entropy scaling, and

      • interfacial properties via classical density function theory.

    • FeOs can use equations of state written as Python classes - perfect for quick and easy prototyping.

    Installation¶

    If you want to use FeOs in Python, you can install it using pip:

    pip install feos

    If you want to use FeOs in Rust, please have a look at the Rust Guide.
