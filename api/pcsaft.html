    PC-SAFT¶

    Implementation of the PC SAFT equation of state.

    Parameter and utility¶

    Identifier

    Create an identifier for a pure substance.

    ChemicalRecord

    Create a chemical record for a pure substance.

    JobackRecord

    Create a set of Joback ideal gas heat capacity parameters for a segment or a pure component.

    PureRecord

    All information required to characterize a pure component.

    SegmentRecord

    All information required to characterize a single segment.

    BinaryRecord

    Create a record for a binary interaction parameter.

    BinarySegmentRecord

    Create a record for a binary segment interaction parameter.

    PcSaftRecord

    Create a set of PC-Saft parameters from records.

    PcSaftParameters

    Create a set of PC-SAFT parameters from records.

    Contributions

    Helmholtz energy contributions to consider when computing a property.

    Verbosity

    Verbosity levels for iterative solvers.
